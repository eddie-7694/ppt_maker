풀-피처 프레젠테이션 시스템

첫 슬라이드입니다. 외부 라이브러리 없이 순수 바닐라로 구현했습니다. 키보드 화살표, 스페이스, PageUp/Down으로 탐색 가능합니다.

빌드 효과 데모

→ 키를 누를 때마다 항목이 등장합니다

  • GPU 가속: transform과 opacity만 애니메이션
  • isAnimating 플래그로 이벤트 잠금/해제
  • data-step 속성으로 순차 등장 제어
  • 빌드 소진 후 다음 슬라이드로 이동
각 항목이 클릭/키 입력마다 하나씩 등장합니다. 모든 항목이 등장한 후 다음 슬라이드로 넘어갑니다. 이전 슬라이드로 돌아오면 빌드가 초기화됩니다.

코드 하이라이트

→ 키로 줄을 순서대로 강조합니다

function goTo(nextIdx, dir) { if (this.locked) return; this.locked = true; const to = this.slides[nextIdx]; to.classList.add('active'); }
코드 블록의 각 줄이 순서대로 강조됩니다. 이미 지나간 줄은 흐리게, 현재 줄은 보라색 강조, 아직 안 된 줄은 더 흐리게 표시됩니다.

핵심 아키텍처

  • Stage: display:flex + 뷰포트 전체 중앙 정렬
  • Deck: aspect-ratio:16/9 + width:min(100vw,177.78vh)
  • Slide: position:absolute; inset:0 스택 구조
  • Font: clamp(min, Nvw, max) 미디어쿼리 없이 반응형
  • Nav: Pointer Events API로 마우스·터치 단일 처리
무대-덱-슬라이드 3단계 구조가 핵심입니다. min() 함수로 가로세로 중 더 제한적인 방향에 맞춰 letterbox/pillarbox가 자동 처리됩니다.
